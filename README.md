# Spotify Website - to listen songs
A basic CRUD Website.

Built With HTML,CSS,JS,SQL

Getting Started -

1.Clone the repository

2.Download XAMP. 

3.Start PHP and My SQL Server. 

4.Upload DB.sql file in SQL.

5.Now you are good to go.
